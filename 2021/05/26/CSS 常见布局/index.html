CSS 常见布局

1.水平垂直居中

1.1行内式

父级元素必须是block，水平居中给父元素添加属性text-align:center

垂直居中：设置本身line-height和height一致即可

1.2 块级，已知宽高

  • 父：relative 子：absolute+left:50%+height:50%+子元素宽高一半的负margin

  • 父：relative 子：absolute+left right top bottom:0+margin:auto

  • 父：relative 子：absolute+left:calc(50%-width/2) +top:calc(50%-height/2)

1.3 块级 宽高未知

  • 父：relative 子：absolute+left:50%+top:50%+transform:translate(-50%,-50%)

  • 父：flex + justify-content:center+align-items:center

  • 父：display:grid 子：justify-self:center align-self:center

2. 两栏布局

  • float+overfolw BFC原理

  • float+margin-left

  • 父：display:flex + 子：flex:1

  • grid

3. 三栏布局 两边定宽中间自适应

  • 圣杯布局

圣杯布局的center是放在最前面的，这样就符合了我们先加载主内容的逻辑

三栏都设置成左浮动 float:left + 父元素设置为左右padding为左右栏的宽度 +对于中间，width：100%+ 对于左栏，position：relative，left：-自身宽度，mergin-left：-100% + 对于右栏position:relative, right：-自身宽度, margin-left：-自身宽度

  • 双飞翼布局

双飞翼布局是在圣杯布局基础上进一步升级来的

盒子排列上，先写包裹中间的父盒子main再写leftright + 对于包裹中间的父盒子main设置float：left，width：100%；对于父盒子里面的实际中间栏，设置margin左右为左右两栏宽度+对于左右两栏也都设置左浮动，左边栏margin-left为-100%+右边栏margin-left为-自身宽度

个人理解 左边栏这个margin-left：-100%就是左中右三栏父容器框宽度的100%

  • float：left float：right+main：overflow：hidden 也就是float+bfc原理

  • flex

  • grid
