JS数组

1.数组基础

js数组是一组有序的数据，和其他语言不一样的是，js数组中的每个槽位都可以存储任意数据类型的数据，数组的长度也是动态的，会随着数据的增删改变。

数组被分成很多个小块的连续内存块，每一个小块都和数字关联，也就是下标，了可以通过这个整数快速的访问对应的小块。并且，数组有length属性，这个属性并不是表示数组的数量，而是表述数组元素的最高序号+1

1.1 数组创建

  • 字面量形式创建

let array=[1,2,3,’a’]

  • new Array()构造函数方式创建

new Array(arg1,arg2,…)参数长度为0或者参数长度大于等于2时，传入的参数按照顺序依次成为数组的元素

new Array(length)参数长度为1且为数值时，构造一个length长的空数组

注意：new可以省略，直接Array()也可以

  • ES6构造器创建

    ES6扩展了数组构造器Array，新增了Array.of和Array.from

    • Array.of 将参数依次转化成数组项，然后返回这个新数组，唯一的用处大概就是Array.of(单个数字)会创建一个一个数字的数组，但是如果是Array()的话就会创造一个数字为length的空数组
    • Array.from 快速基于其他可迭代对象创建新数组，三个参数，第一个参数是一个有迭代器的对象，必选，第二个参数是一个加工函数，新生成的数组会经过该函数的加工载返回，第三个参数时this作用域，也就是加工时执行的this

    注意：用有迭代器的对象除了Array，还有Obj，String，Set, Map

1.2 数组空位

逗号创建空位

ES6会把逗号创建的空位当成存在的元素，只不过值为undefined

而ES5及之前对这个空位进行处理就没用，会当作不存在

1.3 数组索引

通过索引可以获得数组中该索引对应的元素

比较特殊的是，数组的length并不是只读的，可以修改，可以通过修改length来删除数组末尾的元素

数组长度length始终比数组最后一个值的索引大1

2. 数组方法

2.1 复制 填充 方法

2.1.1 fill(value,[start],[end])填充

2.1.2 array.copyWithin(target,[start],[end])

2.2 转化方法

2.2.1 toString()返回由数组中每个值的等效字符串拼接而成的一个逗号分隔的字符串，会对每个值都调用toString()方法，

2.2.2 valueOf()返回数组本身

2.2.3 toLocaleString()，一般情况下与toString()返回相同的结果，区别就是会为每个值都调用toLocalString方法

2.3.4 join()返回一个字符串。先把每个元素都转换成字符串，然后用join中的参数链接各个字符串组合成一个大字符串

2.3 栈方法

push()在数组尾部添加一个元素

pop()在数组尾部弹出一个元素

2.4 队列方法

shift()在数组头部弹出一个元素

unshift()在数组头部添加一个元素

2.5 排序方法

sort([可选参数，是一个比较函数])

sort()默认情况下会为每一个元素调用string转型函数，然后比较字符串来决定顺序，所以可能会出现不准确的情况

如果sort（）里面有一个比较函数的参数，这个参数接收两个参数，第一个参数应该是排在第二个参数的前面就return -1，第一个参数应该排在第二个参数的后面就return 1 ，如果两个参数相等就返回0

reverse()逆序数组中的元素，直接改变原来的数组

2.6 操作方法

  • array.concat(xxx)拼接字符串，不改变原数组，会返回一个新数组

  • array.slice(start,[end]),从已有的数组中返回一个新的选定数组，不改变原数组。start是必须的，指开始位置，如果没有end就默认到数组尾部。如果是复数，就代表倒数第几个，比如-1代表最后一个元素，-2戴白哦倒数第二个元素。

  • array.splice() 很强大 可以向/从数组中添加/删除项目，然后返回被删除的项目，该方法会改变原始数组。

    array.splice(index,howmany,item1,…,itemn)

    • index:必须。整数。规定添加、删除项目的位置，如果是复数就是倒数第几个

    • howmany:必须。要删除的项目数量，如果是0，就不会删除项目。

    • item1…itemn:可选，向数组中添加新的项目。

    • 所以splice()由三种功能：

      • 删除，splice(index,howmany)，传入前两个参数，即要删除的位置和要删除元素的数量
      • 插入，splice(index,0,item…),至少传入三个参数，index插入的位置，删除个数为0，插入的元素item
      • 替换，splice(index,howmany,item..)，至少传入三个参数，开始位置，删除几个，要插入进来的新元素

2.7 归并方法

  • reduce()

    arr.reduce(callback,[initialValue])

    reduce会为数组中的每个元素都调用callback方法，最后会将结果汇总返回单个值

    callback函数有四个参数：

    • pre 上一次回调的返回值
    • currentValue 数组中当前正在被处理的元素
    • index 当前元素在数组中的索引
    • array 调用reduce的数组

    initialValue，作为第一次调用callback的第一个参数,如果没有initialValue，reduce会从所以1的地方才开始执行callback方法，如果有，就从索引0的地方开始执行callback方法

  • reduceRight()和reduce用法一样，只不过是倒序

2.8 搜索位置

  • 按照严格相等===来搜索位置

    • indexOf()从前向后搜索，返回第一个搜索到匹配的位置，搜索不到返回-1
    • lastIndexOf()从后向前搜索，同上
    • includes()返回一个布尔值

  • 按照断言函数搜索数组，每个索引都会调用这个函数，断言函数的返回值决定了是否匹配

    • find() 返回的是第一个匹配的元素，如果没有符合条件的元素返回 undefined；
    • findIndex() 返回的是第一个匹配的元素的索引，如果没有符合条件的元素返回 -1

2.9 迭代器方法

keys(),values(),entries()

2.10 迭代方法

  • forEach() 没有返回值，第一个参数为遍历每个元素的回调函数，第二个参数如果有为要绑定的this
  • map()返回一个新数组，不改变原数组的值
  • filter()过滤数组，满足条件的元素会被返回，返回的是一个新数组，不会改变原来的数组
  • every(),对数组每一项都进行遍历，所有元素都符合条件时，才返回true，否则返回false
  • some(),对数组中每一项都进行遍历，只要有一个元素符合条件就返回true，否则返回false

2.11 for in for of

区别：

  • for…of 遍历获取的是对象的键值，for…in 获取的是对象的键名；

  • for… in 会遍历对象的整个原型链，性能非常差不推荐使用，而 for … of 只遍历当前对象不会遍历原型链；

  • 对于数组的遍历，for…in 会返回数组中所有可枚举的属性(包括原型链上可枚举的属性)，for…of 只返回数组的下标对应的属性值；
